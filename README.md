# Лабораторная работа: Шифрование текста

## Общее описание
Программа реализует два алгоритма шифрования/дешифрования текста:
1. **Метод децимаций** – для текста на **английском языке** (умножение)
2. **Алгоритм Виженера (прямой ключ)** – для текста на **русском языке**

---

## Функциональные возможности
- Ввод текста непосредственно с клавиатуры
- Загрузка текста из текстового файла (.txt)
- Сохранение результата в файл
- Работа с текстом любого размера
- Автоматическое игнорирование символов, не являющихся буквами соответствующего алфавита
- Наглядный графический интерфейс

---

## Реализованные алгоритмы

### 1. Децимация (английский язык)
- **Алфавит:** A–Z (26 букв)
- **Ключ:** целое число, взаимно простое с 26
- **Принцип:** каждая буква заменяется на букву, позиция которой вычисляется умножением позиции исходной буквы на ключ
- **Формула шифрования:** `E(x) = (x * key) mod 26`
- **Формула дешифрования:** `D(x) = (x * key⁻¹) mod 26`, где `key⁻¹` – обратный элемент по модулю 26
- **Особенности:**
  - Ключ нормализуется (`key % 26`)
  - Проверяется взаимная простота ключа с длиной алфавита (GCD(key, 26) = 1)
  - Для дешифрования вычисляется обратный элемент
  - Не-буквенные символы остаются без изменений

### 2. Виженер (русский язык)
- **Алфавит:** А–Я (33 буквы, включая Ё)
- **Ключ:** слово, состоящее из русских букв
- **Принцип:** каждая буква текста сдвигается на величину, соответствующую позиции буквы ключа (ключ повторяется циклически)
- **Формула шифрования:** `C = (P + K) mod 33`
- **Формула дешифрования:** `P = (C - K + 33) mod 33`
- **Особенности:**
  - Ключ очищается от не-букв и приводится к верхнему регистру
  - Не-буквенные символы игнорируются и остаются без изменений
